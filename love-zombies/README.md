LÖVE Game
=========

This is a LÖVE game.

## Dependencies

The project depends on [LÖVE framework](love2d.org) only.

## Compile and Run

```sh
make all  # Builds the `.love` package
make run  # Builds and runs the `.love` package
make      # Defaults to `make run`
```
